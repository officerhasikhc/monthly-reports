🔧 تحريك الحقول (Field Mover)

أداة متقدمة لتحريك حقول التقارير بالسحب والإفلات بسلاسة عالية. اختر التقرير المطلوب وسيتم فتحه مع تفعيل وضع التحريك تلقائياً. يمكنك سحب أي حقل لأي مكان وستظهر الإحداثيات الدقيقة لنسخها للكود.

⌨️ اختصارات لوحة المفاتيح

اختصارات سريعة تعمل داخل أي تقرير:
Ctrl + Shift + M
تبديل وضع التحريك
Ctrl + Z
تراجع (حتى 50 خطوة)
Ctrl + عجلة الماوس
تكبير / تصغير حجم الخط
Ctrl + P
طباعة التقرير

📋 إدارة بيانات الحقول

تعديل أسماء الأشهر، نطاق السنوات، القيم الافتراضية، وجهات الرحلة، وغيرها.

💾 النسخ الاحتياطي والاستعادة

تصدير كل بيانات التقارير إلى ملف JSON أو استيرادها من ملف سابق.

💡 ملاحظات الاستخدام

  • عند تفعيل وضع التحريك، تظهر حدود زرقاء متقطعة ونقاط تحكم (8 مقابض) حول كل حقل
  • نقل الحقل: اسحب من منتصف الحقل بالماوس لتحريكه
  • تغيير الحجم: اسحب أي مقبض (ركن أو حافة) لتكبير/تصغير الحقل - يدعم الطول والعرض وكلاهما معاً
  • شريط أدوات ذكي: يظهر تلقائياً بجوار الحقل بعد النقل أو التحجيم، ويحتوي على:
    • سلايدر حجم الخط - تكبير/تصغير سلس بخطوات 0.5px مع أزرار +/−
    • أزرار المحاذاة - يمين / وسط / يسار
  • حجم الخط السريع: Ctrl + عجلة الماوس فوق الحقل لتكبير/تصغير مباشر
  • تراجع: Ctrl+Z للتراجع عن آخر تحريك أو تغيير حجم (حتى 50 خطوة)
  • أثناء السحب/التحجيم تظهر الإحداثيات في الشريط العلوي + خط محاذاة أفقي
  • بعد الإفلات تظهر لوحة الإحداثيات الكاملة مع زر نسخ
  • انسخ الإحداثيات والصقها في الكود لتثبيت الموضع الجديد
  • لون الإطار: أحمر = نقل، برتقالي = تحجيم، أزرق = تغيير خط
  • الزر العائم 🔓 أو Ctrl+Shift+M لإلغاء وضع التحريك
  • الطباعة تعمل بشكل طبيعي فقط بعد إلغاء وضع التحريك
