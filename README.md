# نظام التقارير الشهرية - المديرية العامة للخدمات الصحية بمحافظة ظفار

## الوصف
نظام ويب لإدارة وتعبئة التقارير الشهرية لمركز حاسك الصحي. يعتمد على صور القوالب الأصلية (PNG) كخلفية مع حقول HTML ذكية تُعبأ تلقائياً.

## التقارير المتاحة
1. **كشف الرواتب الشهرية** (`salary.html`) - كشف رواتب موظفي العقود الخاصة
2. **تقرير خدمات النظافة** (`clean.html`) - تقييم خدمات النظافة أو التغذية
3. **حركة السيارة المستأجرة** (`rent-vehicle.html`) - التشغيل اليومي للسيارة

## بيانات الدخول
- الرقم الوظيفي: `65886`
- المسؤول: عبدالباقي بن عبدالهاي مرزوق بيت مرواس

## الهيكل
```
monthly reports/
├── index.html              # صفحة الدخول + لوحة التقارير
├── salary.html             # كشف الرواتب
├── salary.png              # قالب كشف الرواتب (خلفية)
├── clean.html              # تقرير النظافة
├── clean.png               # قالب تقرير النظافة (خلفية)
├── rent-vehicle.html       # حركة السيارة
├── rent-vehicle.png        # قالب حركة السيارة (خلفية)
├── js/
│   ├── data-cache.js       # طبقة التخزين المؤقت
│   ├── sync-manager.js     # مزامنة البيانات بين التبويبات
│   └── reliable-post.js    # حفظ تلقائي وإرسال موثوق
└── README.md
```

## المميزات
- تعبئة تلقائية للشهر والسنة الحالية
- حفظ البيانات محلياً (localStorage) لكل شهر وسنة
- إدارة بيانات الموظفين (إضافة/تعديل/حذف)
- طباعة مباشرة / تصدير PDF
- فصل كل تقرير عن الآخر لتسهيل الصيانة
- إمكانية إضافة تقارير جديدة مستقبلاً

### شريط الأدوات الاحترافي (v2.0)
- **تصميم حديث**: تدرج لوني احترافي مع أيقونات وقوائم منسدلة
- **حفظ تلقائي**: كل 45 ثانية + حفظ عند إغلاق الصفحة
- **نظام المسودات**: حفظ/تحميل مسودة مؤقتة لكل تقرير
- **نسخ احتياطي**: تصدير/استيراد جميع البيانات كملف JSON
- **طباعة محسّنة**: إخفاء تلقائي لعناصر التحكم
- **وضع مظلم/فاتح**: قابل للتبديل مع حفظ التفضيل
- **تكبير/تصغير**: التحكم بحجم النموذج (50%-200%)
- **شفافية الحقول**: التحكم بشفافية حقول الإدخال
- **اختصارات لوحة المفاتيح**: Ctrl+S حفظ، Ctrl+D مسودة، Ctrl+P طباعة
- **إشعارات ذكية (Toast)**: رسائل نجاح/تحذير/خطأ
- **ساعة + تاريخ هجري**: في شريط الأدوات الثانوي
- **إحصائيات سريعة**: في لوحة التحكم الرئيسية
- **مسح البيانات**: مع تأكيد أمان

## التشغيل
افتح `index.html` في المتصفح أو استخدم أي خادم ويب محلي:
```bash
python -m http.server 5501
```
ثم افتح: http://localhost:5501
